# React Weather UI

A modern, responsive **Weather Application** built with **React** and **Material UI (MUI)** that displays real-time weather information using the **OpenWeatherMap API**.  
The application features a clean UI with dynamic, weather-based visual styling.

---

## Overview

React Weather UI allows users to search for current weather conditions by city name.  
The interface adapts visually based on temperature and humidity, providing an intuitive and engaging user experience.

---

## Features

-  Search weather by **city name**  
-  Display **temperature**, **min/max**, and **feels like**  
-  Show **humidity** levels  
-  **Dynamic card colors** based on weather: Hot / Cold / Rainy  
-  Weather-specific images  
-  **Error handling** for invalid inputs  
-  Fully **responsive and mobile-friendly**  
-  Smooth transitions and hover effects

---

## Technology Stack

- **React** (functional components & hooks)  
- **Material UI (MUI)** for UI components  
- **OpenWeatherMap API** for real-time weather data  
- **CSS Flexbox & Gradients** for styling

## Usage

1. Enter a city name in the search box.
2. Click Search or press Enter.
3. View the weather info card with:
   -Current temperature
   -Min/Max temperature
   -Humidity
   -Description & “feels like” temperature
4. Card styling changes dynamically based on weather.
